The ask, in Sonja's words

"What if two documents or ideas are similar and I want to merge them into one, making sure that what is missing from each when merged is included in the newly merged item."

Decisions already locked (2026-06-05)
  • AI-assisted union — Claude drafts the merged doc; you don't hand-merge.
  • Review before save — the draft opens in an editor; nothing is written until you approve.
  • Sources archived, not deleted — originals are kept (recoverable) and linked to the merged result.

How "lossless" is enforced

The merge prompt instructs Claude to (a) include every distinct fact / point / section from each source, (b) de-duplicate overlapping content, and (c) when two sources contradict, keep both and surface the conflict in a trailing ## Source notes & conflicts section rather than silently picking one. You resolve conflicts during review. Vault entries are never read by Claude, so they cannot be merge sources (OQ3).
